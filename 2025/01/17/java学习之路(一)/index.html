java学习之路(一)

寒假准备开始系统地学习java，因此将学习到的知识记录在blog中，方便随时查看复习。博客的内容大部分来自于博主：沉默王二

一、java语法基础

关键字与保留字：

abstract、boolean、break、byte、case、catch、char、class、continue、default、do、double、else、enum、extends、final、finally、float、for、if、implemnets、import、instanceof、int、interface、long、native、new、null、package、private、protected、public、return、short、static、strictfp、super、switch、synchronized、this、throw、throws、transient、try、void、volatile、while、goto、const

其中：

abstract用于声明抽象类与抽象方法

catch用于捕获try语句中的异常

extends用于指示一个类是从另一个类或接口继承的

final用于表示某个变量、方法或类是最终的，不能被修改或继承

finally和try-catch 配合使用，表示无论是否处理异常，总是执行finally块中的代码

implements用于实现接口

instanceof用于判断对象是否属于某个类型

interface用于声明接口

native用于声明本地方法，本地方法是指在 Java 代码中声明但在本地代码（通常是 C 或 C++ 代码）中实现的方法，它通常用于与操作系统或其他本地库进行交互

new用于创建新对象

packeage用于声明类所在的包

private表示方法或变量只对当前类可见

protected表示方法或变量对同一包内的类和所有子类可见

public表示所有类可见

static表示该变量或方法是静态变量或静态方法

super可用于调用父类的方法或字段

synchronized用于指定多线程代码中的同步方法、变量或代码块

this用于在方法或构造方法中引用当前对象

throw主动抛出异常

throws用于声明异常

transient修饰的字段不会被反序列化

volatile保证不同线程对它修饰的变量进行操作时的可见性，即一个线程修改了某个变量的值，新值对其他线程来说是立即可见的

const用于声明常量

二、java数据类型

前置知识：

java数据类型分为两种：基本数据类型（boolean、char、byte、short、int、long、float 和 double）、引用数据类型（除了基本数据类型以外的类型，都是所谓的引用类型。常见的有数组、class，以及接口）

java变量可分为局部变量、成员变量、静态变量。当变量是局部变量时必须先初始化后才能使用，当变量是成员变量或静态变量时可以不初始化，它们会有默认值

1.boolean:

布尔（boolean）仅用于存储两个值：true 和 false，也就是真和假，通常用于条件的判断。根据 Java 语言规范，boolean 类型只有两个值 true 和 false，但在语言层面，Java 没有明确规定 boolean 类型的大小

2.byte:

一个字节可以表示 2^8 = 256 个不同的值。由于 byte 是有符号的，它的值可以是负数或正数，其取值范围是 -128 到 127（包括 -128 和 127）

3.short:

short 的取值范围在 -32,768 和 32,767 之间，包含 32,767

4.int:

int 的取值范围在 -2,147,483,648（-2 ^ 31）和 2,147,483,647（2 ^ 31 -1）（含）之间。如果没有特殊需求，整型数据就用 int。

5.long:

long 的取值范围在 -9,223,372,036,854,775,808(-2^63) 和 9,223,372,036,854,775,807(2^63 -1)（含）之间。如果 int 存储不下，就用 long。

6.float:

float是单精度的浮点数（单精度浮点数的有效数字大约为 6 到 7 位），32 位（4 字节），遵循 IEEE 754（二进制浮点数算术标准），取值范围为 1.4E-45 到 3.4E+38。float 不适合用于精确的数值

7.char:

char 用于表示 Unicode 字符，占 16 位（2 字节）的存储空间，取值范围为 0 到 65,535

包装器类型：

包装器类型（Wrapper Types）是 Java 中的一种特殊类型，用于将基本数据类型（如 int、float、char 等）转换为对应的对象类型。

Java 提供了以下包装器类型，与基本数据类型一一对应：

  • Byte（对应 byte）
  • Short（对应 short）
  • Integer（对应 int）
  • Long（对应 long）
  • Float（对应 float）
  • Double（对应 double）
  • Character（对应 char）
  • Boolean（对应 boolean）

包装器类型允许我们使用基本数据类型提供的各种实用方法，并兼容需要对象类型的场景。例如，我们可以使用 Integer 类的 parseInt 方法将字符串转换为整数，或使用 Character 类的 isDigit 方法检查字符是否为数字。

基本数据类型和引用数据类型的差别如下：

基本数据类型：

  • 1、变量名指向具体的数值。
  • 2、基本数据类型存储在栈上。

引用数据类型：

  • 1、变量名指向的是存储对象的内存地址，在栈上。
  • 2、内存地址指向的对象存储在堆上。

堆和栈：

堆是在程序运行时在内存中申请的空间（可理解为动态的过程）；切记，不是在编译时；因此，Java 中的对象就放在这里，这样做的好处就是：

栈，能够和处理器（CPU，也就是脑子）直接关联，因此访问速度更快。既然访问速度快，要好好利用啊！Java 就把对象的引用放在栈里。为什么呢？因为引用的使用频率高吗？

不是的，因为 Java 在编译程序时，必须明确的知道存储在栈里的东西的生命周期，否则就没法释放旧的内存来开辟新的内存空间存放引用
